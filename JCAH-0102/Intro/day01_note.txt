	2. Bagaimana komputer memproses sebuah program
		a. Bagian terpenting komputer dalam memproses sebuah aplikasi
			i. Memory : RAM
			ii. CPU : 
		b. Compiler :  
			i. Kelebihan
				1) Siap utk dijalankan
				2) Lebih cepat
				3) Source code tidak terlihat (Blackbox)
			ii. Kekurangan
				1) Tidak lintas platform, hanya utk platform tertentu seperti .exe utk windows dan .app untuk macOS
				2) Tidak fleksibel karena user hanya mendapatkan file hasil compile
				3) Adanya langkah tambahan, yaitu proses compile tadi
			iii. C, C++, Objective-C
		c. Interpreter : 
			i. Kelebihan
				1) Lintas platform, contoh javascript yg bisa dibuka dimana saja selama platform memiliki interpreter(web browser)
				2) Fleksibel, coding yg diberikan masih bisa dirubah
				3) Mudah di uji
			ii. Kekurangan
				1) Butuh interpreter atau web browser
				2) Lebih lambat karena setiap program dijalankan coding harus diinterpretasi dulu
				3) Source code mudah diakses
			iii. Javascript, php
